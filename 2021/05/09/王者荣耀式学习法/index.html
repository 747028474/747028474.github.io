王者荣耀式学习法

发表于2021-05-09|更新于2021-05-10|学习
|阅读量:

前言

首先我们要了解”王者荣耀”的心理学

  1. 明确的目标 : 击杀对手
  2. 清晰的行为路径 : 输出 / 走位 (玩家永远自己在干什么,不迷茫)
  3. 挑战性 : 要击杀对手有一定的难度,但是可以达成
  4. 即使反馈 : 输出 / 走位 中,玩家随时随地的清除操作(结果)
  5. 打卡机制 : 周奖励
  6. 暗示的过程 : 活动任务,弹窗提醒

转化为学习目标

  1. 明确的目标 : 1~6日的学习计划
  2. 清晰的行为路径 : 准备大纲,学习工具等… 辅助类的工具
  3. 挑战性 : 制定一些支线计划, 有大奖励机制
  4. 即使反馈 : 明白自己做完任务有什么神秘的奖励
  5. 打卡机制 : 一周连续打卡小奖励,一月打卡有大奖励
  6. 暗示的过程 : 可以用透明箱子装若干奖励小纸条(例如:奖励手机1小时 , 奖励自己看一场电影…..)

学习过程中,我们要明确目标,定期要有一些奖励机制,有明确记录学习效果,每天通过奖励目标暗示自己学习

分析现状

举例，如果现在说的是高中生，那么你的段位是在钻石，目标是段位达到星耀。

可以参考这个方法测试你的成绩水平：中考前要努力学习，心慌得集中不了注意力，怎么办？

测试结束，你会知道你的成绩水平，知道哪一条路（科目）发育得好不好。

哪些题目不会或者做错，其相关的知识点就是你要学习的地方，也就是查漏补缺。

思路运用

将打王者荣耀的思路运用到学习上的具体方法。

一局王者荣耀比赛是以攻破敌方水晶判定输赢。

一个学期的学习效果是以期末考试成绩判定效果。

这局比赛早已开始，还没到结束阶段，我们还有时间可以争取发育和胜利。

这学期早已开始，还没到学期末，我们还有时间可以争取学习和提高考试成绩水平。

打王者游戏比赛时，你在清兵。有一个敌方英雄干扰你清兵，在降低你的血量。有多个敌方英雄干扰你清兵，还用技能攻击你，血量到底，你的英雄角色也就死亡了。此时，你发育受阻，只能等复活，再清兵发育。

学习过程中，你遇到一个影响你学习的阻碍。如果这个阻碍强度不大，你还是能继续学习。有多个阻碍干扰你学习，你受不了，放弃继续看书、做题、学习。此时，你学习受阻，只能等自己下次想学习，才能继续学习。

当你经济发育起来时，遇到一个或多个敌方英雄都不怕，因为你的英雄经济、血量、技能、连招都足以让你击败敌方英雄并全身而退，更何况还有其他路队友也发育得不错，攻破敌方水晶不成问题。

当你学习的内容越来越多时，那些阻碍都不影响你学习的状态，因为你的学习进度已经足以应付期末考试的题目，更何况你其它考试科目都学得不错，期末成绩的总分就高。

如果你的目标是段位从钻石升到星耀，那就打好钻石小段位每一局排位赛。

如果你的目标是期末考试，那就学习这个学期的各科目考试内容。

如果你的目标是高考，那就学习每个学期的各科目考试内容。

学习方法

想要推塔之前，需要清除敌方小兵。

想要高分通过考试，需要学好每个学习基础单位。也就是说，背诵记忆每一个单词、公式，做对每一道题。

这是最基础的学习方法。

我们打比赛，要看攻略、练英雄技能连招、打红蓝buff、清兵、清高地兵、打我方野怪、打敌方野怪、支援队友打龙等。

我们准备考试，要看学习资料、做题思路、做例题、学基础内容（单词、公式）、学习难点、做练习题、做课外题、看考试重点等。

总结

课业学习跟游戏一样，要博弈、要时间安排、要攻略、要主动学习练习、要找到适合自己的思考方法等等。

在王者荣耀中，不管是地图、游戏设置、游戏角色都是官方提供的，但是我们对角色的理解和使用是不一样，确实是存在攻略，但是用自己的想法打赢一局比赛才是最好的。

在学习上，用更适合你的理解和学习方法，会取得更好的成绩，也就是最好的将打王者荣耀的思路运用到学习上。
